# 모두의주일학교 - 학생 가이드

## 1. 시작하기

### 로그인 방법
1. **modoosunday.com** 접속
2. **교회 코드** 입력 (선생님께 문의)
3. **로그인 코드** 입력 (6자리 숫자)
4. **로그인** 버튼 클릭

---

## 2. 대시보드

로그인 후 대시보드에서 확인할 수 있는 것들:

### 내 정보
- 이름과 소속 팀
- 현재 보유 달란트

### 이번 주 암송 말씀
- 한국어/영어/프랑스어로 볼 수 있음
- 언어 버튼을 눌러 전환

### 이번 주 퀘스트
- **출석**: 일요일 예배 참석 시 체크됨
- **암송**: 일요일에 암송 성공 시 체크됨
- **QT**: 월~토 매일 큐티 인증 시 체크됨 (총 6개)

### 달력
- 이번 달 퀘스트 기록을 한눈에 확인
- 점으로 표시: 출석(파랑), 암송(초록), QT(노랑)

### 이번 주 요약
- 출석, 암송, QT 완료 횟수
- 이번 주 획득 달란트

---

## 3. QT 인증하기

### QT란?
QT(Quiet Time)는 매일 말씀을 읽고 묵상하는 시간입니다.

### 인증 방법
1. 하단 메뉴에서 **QT 인증** 탭 클릭
2. **오늘의 QT 주제** 확인
3. QT 후 **사진 촬영** 또는 **갤러리에서 선택**
4. **업로드** 버튼 클릭
5. 선생님 승인 대기

### 상태
- **미제출**: 아직 인증 안 함
- **승인 대기**: 선생님 확인 중
- **승인 완료**: 달란트 지급됨

### 주의사항
- 하루에 한 번만 인증 가능 (월~토)
- 일요일은 QT 인증 없음 (예배 출석!)
- 사진이 선명해야 승인됨

---

## 4. 리더보드

### 확인 방법
1. 하단 메뉴에서 **리더보드** 탭 클릭

### 표시 내용
- 전체 학생 달란트 순위
- 상위 3명은 메달 표시 (금/은/동)
- 내 순위는 파란색으로 강조

---

## 5. 내 팀

### 확인 방법
1. 하단 메뉴에서 **내 팀** 탭 클릭

### 표시 내용
- 팀 이름과 색상
- 팀원 목록

---

## 6. 달란트 시스템

### 달란트란?
주일학교 활동에 참여하면 받는 포인트입니다.

### 획득 방법
| 활동 | 달란트 |
|------|--------|
| 출석 | 10 달란트 |
| 암송 | 20 달란트 |
| QT (1회) | 15 달란트 |

*달란트 수치는 교회마다 다를 수 있습니다.

### 사용 방법
- 선생님께서 정해주신 방법으로 사용 가능
- 예: 간식 교환, 선물 교환 등

---

## 7. 앱 설치하기 (홈 화면 추가)

### 아이폰
1. Safari로 modoosunday.com 접속
2. 하단 **공유 버튼** (네모에서 화살표) 클릭
3. **홈 화면에 추가** 선택
4. **추가** 클릭

### 안드로이드
1. Chrome으로 modoosunday.com 접속
2. 우측 상단 **메뉴** (점 3개) 클릭
3. **홈 화면에 추가** 또는 **앱 설치** 선택
4. **설치** 클릭

## 8. 자주 묻는 질문

### Q: 로그인 코드를 잊어버렸어요
**A**: 선생님께 문의하세요.

### Q: QT 인증이 안 돼요
**A**:
- 인터넷 연결 확인
- 사진 파일 크기가 너무 큰지 확인
- 이미 오늘 인증했는지 확인

### Q: 달란트가 안 올랐어요
**A**: 선생님이 승인하면 자동으로 올라갑니다. 잠시 기다려주세요.

### Q: 다른 사람 달란트가 보이나요?
**A**: 리더보드에서 순위만 확인 가능합니다.
