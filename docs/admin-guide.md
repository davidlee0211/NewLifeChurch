# 모두의주일학교 - 관리자(교사) 가이드

## 1. 시작하기

### 로그인 방법
1. **modoosunday.com** 접속
2. **교회 코드** 입력
3. **관리자 아이디** 입력 (문자)
4. **로그인** 버튼 클릭

---

## 2. 대시보드 개요

로그인 후 관리자 대시보드에서 한눈에 확인할 수 있는 것들:

### 상단 요약 카드
- **이번 주 출석**: 출석한 학생 수 / 전체 학생 수
- **이번 주 암송**: 암송한 학생 수 / 전체 학생 수
- **오늘 QT 제출**: 오늘 QT 인증한 학생 수
- **승인 대기**: 승인 대기 중인 QT 수

### 빠른 실행 버튼
- 출석/암송 체크
- QT 주제 등록
- QT 승인
- 달란트 관리

### 하단 정보
- 학생 달란트 현황 (Top 3)
- 최근 QT 제출 목록

---

## 3. 출석/암송 체크

### 접근 방법
좌측 메뉴 → **출석/암송** 또는 대시보드 빠른 실행 버튼

### 사용 방법
1. **주 선택**: 상단 화살표로 이전/다음 주 이동
2. **학생 목록** 확인
3. **출석** 체크박스: 예배 참석한 학생 체크
4. **암송** 체크박스: 암송 성공한 학생 체크
5. **저장** 버튼 클릭

### 통계 표시
- 출석: N명
- 결석: N명
- 암송: N명
- 전체: N명

### 참고사항
- 체크 시 자동으로 달란트 지급
- 체크 해제 시 자동으로 달란트 차감
- 과거 주차도 수정 가능

---

## 4. QT 승인

### 접근 방법
좌측 메뉴 → **QT 승인** 또는 대시보드 빠른 실행 버튼

### 화면 구성
- **좌측**: 승인 대기 중인 QT 목록
- **우측**: 선택한 QT 상세 정보

### 사용 방법
1. 좌측 목록에서 **학생 선택**
2. 우측에서 **사진 확인** (클릭하면 확대)
3. **승인** 또는 **반려** 버튼 클릭

### 승인 기준 (권장)
- QT 노트 또는 성경 사진이 선명한가
- 날짜가 맞는가
- 본인이 작성한 것인가

### 참고사항
- 승인 시 자동으로 달란트 지급
- 반려 시 학생이 다시 제출 가능

---

## 5. QT 주제 등록

### 접근 방법
좌측 메뉴 → **QT 주제**

### 사용 방법
1. **날짜 선택** (월~토)
2. **제목** 입력 (예: "마태복음 5:1-12")
3. **내용** 입력 (묵상 가이드)
4. **이미지 업로드** (선택사항)
5. **저장** 버튼 클릭

### 참고사항
- 학생이 QT 인증 화면에서 오늘의 주제를 볼 수 있음
- 일요일은 QT가 없으므로 등록 불필요

---

## 6. 암송 말씀 등록

### 접근 방법
좌측 메뉴 → **암송 말씀**

### 사용 방법
1. **주 시작일** 선택
2. **성경 구절** 입력
   - 한국어 (필수)
   - 영어 (선택)
   - 프랑스어 (선택)
3. **저장** 버튼 클릭

### 참고사항
- 학생 대시보드에서 이번 주 암송 말씀이 표시됨
- 다국어 지원으로 외국인 학생도 사용 가능

---

## 7. 달란트 관리

### 접근 방법
좌측 메뉴 → **달란트 관리**

### 사용 방법
1. **학생 선택** (드롭다운 또는 검색)
2. **달란트 수량** 입력
3. **사유** 입력 (예: "특별 활동 참여")
4. **지급** 또는 **차감** 버튼 클릭

### 활용 예시
- 특별 활동 참여 보상
- 봉사 활동 보상
- 규칙 위반 시 차감
- 이벤트 상품 교환 시 차감

### 최근 거래 내역
- 하단에서 최근 달란트 거래 확인 가능

---

## 8. 학생 관리

### 접근 방법
좌측 메뉴 → **학생 관리**

### 학생 등록
1. **학생 추가** 버튼 클릭
2. **이름** 입력
3. **로그인 코드** 입력 (6자리 숫자, 중복 불가)
4. **팀 선택**
5. **저장** 버튼 클릭

### 학생 수정
1. 학생 목록에서 **수정** 버튼 클릭
2. 정보 수정 후 **저장**

### 학생 삭제
1. 학생 목록에서 **삭제** 버튼 클릭
2. 확인 후 삭제

### 참고사항
- 로그인 코드는 학생에게 알려주세요
- 팀은 설정에서 추가/수정 가능

---

## 9. 게임

### 접근 방법
좌측 메뉴 → **게임**

### 팀 뽑기
- 랜덤으로 학생을 팀에 배정
- 새 학기 팀 구성 시 활용

### 퀴즈 보드
- 등록된 퀴즈로 실시간 게임 진행
- 팀 대항전 가능

---

## 10. 퀴즈 관리 (바이블다이스)

### 접근 방법
좌측 메뉴 → **퀴즈**

### 퀴즈 형식
- **객관식 (4지선다)**: 보기 4개 중 정답 선택
- **주관식**: 학생이 직접 답변, 교사가 채점

### 객관식 퀴즈 등록
1. **새 퀴즈 추가** 버튼 클릭
2. **퀴즈 유형**: 객관식 (4지선다) 선택
3. **문제** 입력 (예: "노아의 방주에 들어간 동물은 몇 쌍씩일까요?")
4. **보기 1~4** 입력
5. **정답 선택** (1~4 중 정답 번호 클릭)
6. **퀴즈 저장** 버튼 클릭

### 주관식 퀴즈 등록
1. **새 퀴즈 추가** 버튼 클릭
2. **퀴즈 유형**: 주관식 선택
3. **문제** 입력 (예: "예수님의 열두 제자 중 세리였던 사람은?")
4. **정답** 입력 (예: "마태")
5. **퀴즈 저장** 버튼 클릭

### 게임 중 진행
- **객관식**: 4개 보기 화면에 표시 → 정답 공개 → 팀 선택
- **주관식**: 문제만 표시 → 학생 답변 → 정답 공개 → 교사가 맞춘 팀 선택

### 퀴즈 관리
- **활성/비활성**: 게임에서 사용 여부 설정
- **수정**: 문제, 보기, 정답 수정
- **삭제**: 퀴즈 삭제

### 활용
- 바이블다이스 게임에서 사용
- 성경 퀴즈, 암송 퀴즈 등

좌측 메뉴 → **설정**

### 교회 정보
- 교회 이름 수정
- 로고 변경

### 달란트 설정
각 퀘스트별 지급 달란트 수정:
- 출석: 기본 10
- 암송: 기본 20
- QT: 기본 15

### 팀 관리
- 팀 추가
- 팀 이름/색상 수정
- 팀 삭제

## 13. 자주 묻는 질문

### Q: 학생이 로그인이 안 된다고 해요
**A**:
- 교회 코드가 정확한지 확인
- 로그인 코드(6자리 숫자)가 정확한지 확인
- 학생 관리에서 해당 학생이 등록되어 있는지 확인

### Q: QT 승인했는데 달란트가 안 올랐어요
**A**:
- 페이지 새로고침 후 확인
- 달란트 설정에서 QT 달란트가 0이 아닌지 확인

### Q: 지난 주 출석을 수정하고 싶어요
**A**: 출석/암송 페이지에서 이전 주로 이동하여 수정 가능합니다.

### Q: 학생 로그인 코드를 변경하고 싶어요
**A**: 학생 관리 → 해당 학생 수정에서 변경 가능합니다.

### Q: 팀을 추가하고 싶어요
**A**: 설정 → 팀 관리에서 추가 가능합니다.
